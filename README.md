Student Registration System.

Description.

This is a simple Student Registration System built using HTML, CSS, and JavaScript. It allows users to add student details, display them, and clear the output.

Features.

* Add student name, ID, and marks
* Display student records
* Clear displayed data
* Simple and user-friendly interface

Technologies Used.

* HTML
* CSS
* JavaScript

Project Structure.

* index.html → Contains HTML, CSS, and JavaScript code

How to Run.

1. Download or clone the repository
2. Open `index.html` in any web browser
3. Start adding student data
   
 Future Improvements.

* Save data permanently (Local Storage)
* Add edit/delete options
* Improve UI design
